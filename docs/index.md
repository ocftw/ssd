# :material-security: 資安防護基礎教材

本教材協助您與您的組織在資訊安全的增進，課程收錄九大部分，各部分依序提供基本的教材、課程章節中提及的安全參作手冊、組織內政策制定範本、組織狀態評估。

陪伴教材各章節以可自主學習、由淺入深、手把手的原則方式進行編寫，如有課程上的疑問，也可透過關懷時間詢問與解答。
